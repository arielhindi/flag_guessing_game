## Game Flow
- The GUI displays a large flag emoji.
- Four country names appear as clickable buttons.
- Click the correct country name.
- The game keeps score and displays the final accuracy when you click Stop.

## Dependencies
- Python 3.8+
- tkinter (standard library)
- No external packages required!

## Install and run
From the repository root:

```bash
python -m day05.ui
```

## Tests
Run the test suite from the repository root:

```bash
pytest -q
```
